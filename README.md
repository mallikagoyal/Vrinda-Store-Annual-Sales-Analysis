# 📊 Vrinda Store Annual Sales Analysis 2022

## Project Overview
This project analyzes the annual sales performance of Vrinda Store for the year 2022 using Microsoft Excel. The dashboard provides insights into sales trends, customer demographics, order status, regional performance, and sales channels.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Key Insights
- Women contributed approximately 64% of total sales.
- Over 90% of orders were successfully delivered.
- Maharashtra was the highest revenue-generating state.
- Amazon was the leading sales channel.
- Adult customers generated the majority of orders.

## Dashboard Features
- Orders vs Sales Analysis
- Sales by Gender
- Order Status Analysis
- Top 5 States by Sales
- Age vs Gender Analysis
- Sales Channel Performance

## Business Recommendations
- Focus marketing efforts on female customers.
- Strengthen presence in top-performing states.
- Improve customer retention strategies.
- Increase investment in high-performing sales channels.
